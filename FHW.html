Fundamentos de Hardware

En esta asignatura aprenderemos a:

  • Utilizar la placa Arduino que es una tarjeta elecrtónica programable que permite la creación rápida de prototipos
  • Programar en Processing.
  • Configuración de equipos y periféricos y la Arquitectura de Ordenadores.
  • Instalación de Software de Utilidad y Propósito General para un Sistema Informático.
  • Creación de Imágenes de Software. Respaldo de Software base de un sistema.
